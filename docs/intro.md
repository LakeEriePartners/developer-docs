# TPA Stream Developer Documentation

Welcome to TPA Stream. This site is the home for everything you need
to integrate with the TPA Stream platform: the REST API, real-time
webhooks, and the JavaScript Connect SDK that embeds member-side
enrollment into your own application.

If you're planning to use the API in production, please review the
[Privacy Policy](https://app.tpastream.com/privacy).

## The three integration surfaces

TPA Stream's developer tools fall into three areas, each with its own
section in the sidebar:

- **[Connect SDK](/connect/overview)** — a JavaScript client library
  that you embed into your web or mobile application so members can
  link their health insurance accounts to your system.
- **[Webhooks](/connect/webhooks-claim)** — TPA Stream POSTs your
  application in real time as new claims and crawl results arrive.
- **[REST API](/api/overview)** — pull-based access to claims, policy
  holders, payers, and the rest of the data model.

For a guided walkthrough, start with [Getting Started](/getting-started).
